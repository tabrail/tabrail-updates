## Static Release Notes

Release notes are published as static files under:

- `https://updates.tabrail.com/release-notes/<version>/en.html`
- `https://updates.tabrail.com/release-notes/<version>/zh-Hans.html`
- `https://updates.tabrail.com/release-notes/<version>/ja.html`
- `https://updates.tabrail.com/release-notes/<version>/ko.html`

Example:

- `https://updates.tabrail.com/release-notes/0.0.7/en.html`

## Pages Mode

GitHub Pages should be set to deploy from `main` branch root (`/`). In this mode,
GitHub renders this `README.md` as the homepage, matching the vibe-island-updates setup.
